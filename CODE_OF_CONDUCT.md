# Contributor Covenant 行動規範

## 前文

ClipSave は、オープンで革新的かつインクルーシブなコミュニティを目指します。  
このコミュニティで期待される振る舞いを明確にするため、Contributor Covenant（.NET Foundation の採用方針を参考）に基づく本行動規範を採用します。

## 私たちの約束

私たちメンバー、コントリビューター、リーダーは、年齢、体格、可視・不可視の障害、民族、性徴、性自認とジェンダー表現、経験レベル、学歴、社会経済的地位、国籍、外見、人種、宗教、性的指向にかかわらず、誰にとってもハラスメントのない参加体験を提供することを約束します。

また、オープンで歓迎的、多様で包摂的、そして健全なコミュニティづくりにつながる行動と対話を行うことを約束します。

## 私たちの基準

コミュニティにとって望ましい行動の例:

- 他者への共感と親切を示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、また礼節をもって受け入れる
- 自分の過ちに責任を持ち、影響を受けた人に謝罪し、そこから学ぶ
- 個人の利益だけでなく、コミュニティ全体にとって最善となる判断を行う

コミュニティにとって望ましくない行動の例:

- 性的な言動・画像の使用、またはあらゆる形の性的接近
- 荒らし行為、侮辱的・中傷的コメント、個人攻撃や政治的攻撃
- 公私を問わないハラスメント
- 明示的な許可なく、他者の個人情報（住所・メールアドレス等）を公開する行為
- その他、職業的な場において不適切と合理的に判断される行為

## 執行責任

コミュニティリーダーは、許容される行動基準を明確化し、これを公正に執行する責任を負います。  
不適切、脅迫的、攻撃的、有害と判断される行為に対しては、適切かつ公正な是正措置を講じます。

コミュニティリーダーは、本行動規範に沿わないコメント、コミット、コード、Wiki 編集、Issue、その他のコントリビューションを削除・編集・却下する権限と責任を持ち、必要に応じてその判断理由を説明します。

## 適用範囲

本行動規範は、すべてのコミュニティ空間に適用されます。  
また、個人がコミュニティの公式な代表として公の場で行動する場合にも適用されます。

代表行為の例:

- 公式メールアドレスの使用
- 公式ソーシャルメディアアカウントでの投稿
- オンライン／オフラインイベントで任命された代表としての活動

## 施行

ハラスメント、虐待、その他の受け入れられない行為は、執行担当へ報告してください。  
報告先: `tnagata012@gmail.com`

すべての報告は迅速かつ公正に確認・調査されます。  
執行担当は、報告者のプライバシーと安全を尊重する義務を負います。

## 執行ガイドライン

コミュニティリーダーは、違反に対する対応を決定する際、次のコミュニティ影響ガイドラインに従います。

### 1. Correction（是正）

コミュニティ影響:
不適切な言葉遣い、またはコミュニティ内で専門性を欠く・歓迎されないと判断される行為。

対応:
違反の内容と不適切である理由を説明した、非公開の書面による警告。  
必要に応じて公開謝罪を求める場合があります。

### 2. Warning（警告）

コミュニティ影響:
単発、または複数回にわたる違反行為。

対応:
継続時の措置を明示した警告。  
指定期間、関係者および執行担当者への接触（要求されていない接触を含む）を禁止します。  
この禁止は、コミュニティ空間だけでなく外部チャネル（SNS 等）にも及びます。  
違反が継続した場合、一時的または永久的な参加停止につながる可能性があります。

### 3. Temporary Ban（一時的参加停止）

コミュニティ影響:
継続的な不適切行為を含む、重大な基準違反。

対応:
指定期間、コミュニティとのあらゆる対話・公開コミュニケーションを禁止します。  
期間中、関係者および執行担当者への公私いずれの接触も認められません。  
違反した場合、永久的な参加停止につながる可能性があります。

### 4. Permanent Ban（永久的参加停止）

コミュニティ影響:
継続的な不適切行為、個人へのハラスメント、または特定属性集団への攻撃・蔑視など、基準違反の反復パターンが認められる場合。

対応:
コミュニティ内でのあらゆる公開的な対話への永久的な参加停止。
