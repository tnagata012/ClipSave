# 製品コンセプト

**このドキュメントの目的**: ClipSave のビジョン、設計思想、製品の方向性を定義します。

## ビジョン

> クリップボードのコンテンツを、最小限の操作で素早くファイルに保存できるツールを提供する。

「コピー → `Ctrl+Shift+V`」の 2 ステップで完結。それ以上の操作は不要。

## 設計思想

### シンプルさを追求

- **ワンアクション保存**: ホットキー一発で保存完了
- **自動判別**: ユーザーがコンテンツ種別を意識する必要なし
- **ウィンドウレス**: 常駐してもデスクトップを邪魔しない
- **迷わない起動体験**: 自動起動は既定で有効、変更は Windows スタートアップ設定へ 1 クリックで到達可能

### プライバシー最優先

- **ネットワーク通信なし**: データは一切外部に送信されない
- **テレメトリなし**: 利用状況の収集を行わない
- **ローカル完結**: すべてのデータはユーザーの PC 内で処理

### クリーン

- **アンインストールで完全消去**: ログ・設定・クラッシュダンプなど全アプリデータを MSIX 管理下に集約し、削除時に痕跡を残さない
- **システムを汚さない**: レジストリ・環境変数への書き込みなしで、スタートアップも MSIX マニフェストで管理

### 軽量・省リソース

- **イベント駆動**: ポーリングなし、CPU 使用率ほぼゼロ
- **最小限の機能**: 画像編集、履歴管理、クラウド同期は対象外

## 対象ユーザー

- スクリーンショットを頻繁に保存する人
- コードスニペットやログをファイルに残したい開発者
- コピーした表データを素早く CSV 化したい人

## 対応コンテンツ

| 種別 | 拡張子 | ユースケース |
|------|--------|-------------|
| 画像 | `.png` / `.jpg` | スクリーンショット、コピーした図 |
| 表データ | `.csv` | Excel や Web からコピーした表 |
| JSON | `.json` | API レスポンス、設定データ |
| Markdown | `.md` | ドキュメント、メモ |
| テキスト | `.txt` | その他のプレーンテキスト |

詳細な判別ルールは [使い方ガイド](UsageGuide.md) を参照してください。

## 配布形態

| チャネル | 形式 | 対象 |
|---------|------|------|
| Microsoft Store | MSIX（無料） | 一般ユーザー |
| GitHub Releases | MSIX | 開発版の試用、オフライン環境 |
| ソースコード | OSS | 開発者、カスタマイズ |

## 非機能要件

| 項目 | 要件 |
|------|------|
| 対応 OS | Windows 11 |
| ランタイム | .NET 10 |
| パフォーマンス | CPU 使用率ほぼゼロ、ポーリング禁止 |
| セキュリティ | ネットワーク通信なし、テレメトリなし |
