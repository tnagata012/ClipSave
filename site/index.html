主な機能

コピーしたものを、自動で最適なフォーマットに。

画像

スクリーンショットやコピーした画像をPNG / JPGで保存。PNGは透過を維持し、JPGは白背景で保存。

テキスト

プレーンテキストをUTF-8 (BOMなし) で保存。コードスニペットの退避に。

JSON

JSON文字列を自動検出し、インデント整形して保存。APIレスポンスの確認に。

CSV

タブ区切りの表データを検出し、CSV形式に変換して保存。Excel連携もスムーズ。

Markdown

見出し・リスト・コードブロックを検出してMarkdownとして保存。メモの即時保存に。

自動判別

クリップボードの中身を自動で解析し、最適な形式を選択。手動設定は不要。

使い方

たった3ステップ。設定すら不要。

1

コピーする

テキストや画像をいつも通りCtrl+Cでコピー。

2

ホットキーを押す

Ctrl+Shift+Vを押すだけ。ClipSaveがバックグラウンドで即座に処理。

3

保存完了

デスクトップまたは Explorer フォルダに即保存。通知でお知らせ。

詳細

細部まで使いやすさにこだわった設計。

ファイル命名

プレフィックス_yyyyMMdd_HHmmss.拡張子 の形式で自動命名（既定: CS）。タイムスタンプの有無も切り替え可能。重複時は連番 _1, _2 を付与（プレフィックスなし・タイムスタンプなしのときは 1, 2...）。

保存先

デスクトップがアクティブなときはデスクトップへ保存。Explorerがアクティブで実フォルダ表示中なら、そのフォルダへ保存（This PC・ごみ箱などの仮想フォルダは対象外）。

カスタマイズ

画像形式 (PNG/JPG)、JPG品質、ファイル名プレフィックス、タイムスタンプ付与、ホットキー変更、コンテンツ種別の有効/無効、通知設定をGUIで。

通知

保存成功・空クリップボード・エラーの各通知を個別にオン/オフ可能。OS通知設定も尊重。

プライバシー

ネットワーク通信・テレメトリは一切なし。データはすべてローカルで完結。アンインストール時にアプリデータを完全削除。

よくある質問

ClipSave は無料ですか？

はい、完全に無料です。MIT License のオープンソースソフトウェアとして公開しています。

ホットキーは変更できますか？

はい、設定画面からお好みのキーの組み合わせに変更できます。

画像の保存形式は選べますか？

PNG と JPG から選択できます。JPG の場合は品質（1〜100）も設定可能です。

保存先はどこになりますか？

デスクトップがアクティブなときはデスクトップに、Explorer でフォルダを開いているときはそのフォルダに保存されます。

クリップボードが空のときはどうなりますか？

ファイルは作成されません。通知は設定で有効な場合のみ表示されます（既定ではオフ）。

Windows 起動時に自動で起動できますか？

はい、MSIX パッケージとしてインストールすると、スタートアップに自動登録されます。

ClipSave を始めよう

インストールして、Ctrl+Shift+V を押すだけ。
